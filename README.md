# VATSIM.js

A small webapp for viewing vatsim traffic - part of an assignment in Web Applications at Hi√ò, written with love in TS <3

## Setup

Get the code.

```bash
git clone https://github.com/olejorga/vatsim.js.git
cd vatsim.js
```

To start the app, run the development server.

```bash
npm run dev
```

## How to use

Open [http://localhost:3000](http://localhost:3000) with your browser to use the app.
